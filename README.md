# wechat-fun

## V1.0特性

>> 注：本项目因便捷工作而写，不涉及到任何商业用途，产生的后果请自行承担。

1. 根据关键字自动加好友
2. 转发群里某人关键字信息，便于整理PM
3. 自助聊天机器人（机器人API Key，可替换）


## 需要
- Python 3
- wxpy
- platform
- random

##安装
下载源码
'''
https://github.com/zhuyouzhi1314/wechat-fun.git
'''

'''
安装拓展

cd wechat-fun
pip3 install wxpy
'''

'''
主程序执行

python3 main.py
'''
